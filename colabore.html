Colabore com o Mapa

Este projeto tem código aberto e oferece amplo espaço para a colaboração dos usuários e interessados. Por isso, é bem-vindo todo esforço para ampliar a qualidade, as funcionalidades, o alcance e os serviços públicos prestados pelo portal. A lista abaixo indica algumas ações que podem ser realizadas por indivíduos e instituições. Você pode sugerir outras. Se houver interesse, entre em contato.

Teremos o prazer de dar os merecidos créditos para todos os colaboradores voluntários do Mapa.

  1. Traduzir este Mapa para outros idiomas e facilitar o consumo dos dados em outros países. Nosso objetivo é ter o Mapa traduzido pelo menos para os seguintes idiomas:
    1. Inglês
    2. Espanhol
    3. Francês
    4. Alemão
  2. Colaborar para identificar e conectar ao Mapa bases de dados de parcerias celebradas entre OSCs e governos municipais ou entre OSCs e outras organizações privadas. Um dos maiores desafios deste portal é a obtenção de dados atualizados e confiáveis sobre as parcerias das OSCs com o Poder Público em seus diversos níveis ou com outras organizações privadas (sejam ou não entidades sem fins lucrativos). Se você é um desenvolvedor, pode apoiar o Mapa na construção de serviços de dados em algum município, que facilite a conexão e envio das informações para o Mapa. Ou pode ainda colaborar na identificação de serviços de dados abertos que possam ser consumidos pelo Mapa (os dados devem atender aos requisitos indicados neste documento.(PDF 285KB)).
  3. Desenvolver aplicativos de interesse público com base nos dados do Mapa das Organizações da Sociedade Civil.
  4. Sugerir aprimoramentos ao código de tratamento dos dados do Mapa. Programadores e pesquisadores que trabalhem com o pacote estatístico R podem acompanhar o desenvolvimento dos códigos de limpeza e tratamento das bases do Mapa, bem como propor melhorias a esses códigos. As sugestões de mudanças nos códigos podem ser feitas na própria ferramenta (ver link acima) que hospeda os códigos feitos para o portal.
  5. Colaborar na construção do Portal
  6. Tornar-se um apoiador institucional para manter a gestão e o desenvolvimento do portal.
