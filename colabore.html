Colabore com o Mapa

Este projeto tem código aberto e há amplo espaço para a colaboração dos usuários e interessados. Por isso, é bem-vindo todo esforço para ampliar a qualidade, as funcionalidades, o alcance e os serviços públicos prestados pelo portal. A lista abaixo indica algumas ações que podem ser realizadas por indivíduos e instituições. Você pode sugerir outras. Se houver interesse, entre em contato.

  1. Traduzir este Mapa para outros idiomas e facilitar o consumo dos dados em outros países. Nosso objetivo é ter o Mapa traduzido pelo menos nos seguintes idiomas:
    1. Inglês
    2. Espanhol
    3. Francês
    4. Alemão
  2. Colaborar para identificar e conectar no Mapa bases de dados de parcerias celebradas entre governos municipais e OSCs. Um dos maiores desafios deste portal é obter dados atuais, contínuos e seguros sobre parcerias com OSCs realizadas pelos entes federados. Se você é um desenvolvedor, pode apoiar na construção de serviços de dados em algum município, que facilite a conexão e envio das informações para o Mapa. Ou colaborar para identificar onde há dados abertos, e com os campos mínimos solicitados pelo Mapa para que as bases possam ser consumidas por nós.
  3. Desenvolver aplicativos de interesse público, com base nos dados do portal.
  4. Colaborar na construção do Mapa
  5. Tornar-se um apoiador institucional para manter em funcionamento da equipe de desenvolvimento do portal.
