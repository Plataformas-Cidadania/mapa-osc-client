Base de Dados

O Mapa disponibiliza mensalmente para download as bases de dados separadas em categorias. As bases abaixo foram extraídas no mês de .

  • Base de dados dos recursos das OSCs

    Para dar ampla transparência e garantir o acesso à informação contida no mapa em relação aos recursos das OSCs esta seção do Mapa das Organizações da Sociedade Civil disponibiliza as informações referente a declaração de cada OSC sobre seus recursos em formato apropriado para download e armazenamento. O usuário poderá baixar as informações para fazer todos os cruzamentos e análises que desejar e realizar estudos e pesquisas a partir desses dados.

    Baixar a base de dados dos recursos das OSCs
